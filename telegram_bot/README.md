# Telegram бот для генерации поздравительных изображений

Простой Telegram бот для генерации персонализированных поздравительных изображений через GigaChat API.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте бота через @BotFather в Telegram:
   - Откройте Telegram и найдите @BotFather
   - Отправьте команду `/newbot`
   - Следуйте инструкциям для создания бота
   - Скопируйте полученный токен

3. Добавьте токен в `.env` файл:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
GIGACHAT_CREDENTIALS=ваш_ключ_gigachat
```

## Запуск

```bash
python telegram_bot/bot.py
```

## Использование

1. Найдите вашего бота в Telegram
2. Отправьте команду `/start` для приветствия
3. Отправьте команду `/generate` чтобы увидеть формат ввода данных
4. Отправьте все данные одним сообщением, разделяя их переносами строк:
   ```
   01.01.2025
   Иван Петров
   ООО 'ТехноСтрой'
   Генеральный директор
   VIP
   официальный
   премиум качество, корпоративный стиль
   кредитная линия
   ```
5. Бот сгенерирует и отправит изображение

## Формат данных

Отправьте все данные одним сообщением (8 строк, разделенных переносами):

1. **Дата события** (обязательно, формат DD.MM.YYYY)
2. **Имя клиента** (опционально, можно оставить пустым)
3. **Название компании** (опционально)
4. **Должность** (опционально)
5. **Сегмент клиента** (опционально: VIP, новый, лояльный, стандартный)
6. **Тон** (опционально: официальный, дружеский, креативный)
7. **Предпочтения** (опционально, через запятую)
8. **Тема истории взаимодействий** (опционально)

Если какое-то поле не нужно, просто оставьте пустую строку.

## Команды

- `/start` - приветствие и инструкции
- `/generate` - показать формат ввода данных

## Примеры дат

- `01.01.2025` - Новый год (определится автоматически)
- `08.03.2025` - 8 Марта (определится автоматически)
- `15.03.2025` - День рождения (определится автоматически)

Тип события определяется автоматически по дате.
